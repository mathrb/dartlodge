# X01 (Standard Darts)

## Overview
The most popular darts game format, commonly called "501" but actually a family of games (301, 501, 701, etc.). Players reduce their score to exactly zero according to configured in/out strategies.

## Game Configuration

### Basic Settings
- **Number of Rounds**: 15, 30, 50, or 80 (default: 30)
- **Starting Score**: 301, 501 (default, championship standard), 701, 901

### In Strategies (Starting Requirements)
- **Straight In**: All darts count immediately
- **Double In**: First dart must hit a double to start scoring
- **Master In**: First dart must hit double or triple to start scoring

### Out Strategies (Finishing Requirements)
- **Straight Out**: Finish on any segment
- **Double Out**: Final dart must be a double (championship standard)
- **Master Out**: Final dart must be double or triple

### Players

1 to 6 players can join the game. A handicap can be assigned to any player:

* -50 (ie: for a 301 game, start at 251)
* -100 (ie: for a 301 game, start at 201)
* -150 (ie: for a 301 game, start at 151)
* -200 (ie: for a 301 game, start at 101)

## Rules

### Core Gameplay
- **Darts per Turn**: 3 darts
- **Objective**: Reduce score to exactly zero
- **Bust**: Score goes below zero or to one → turn ends, score reverts
- **Leg Winner**: First to reach exactly zero with valid out strategy
- **Match Winner**: First to win configured number of legs

### In Strategy Details
- Applies only to first dart of each leg
- If failed: dart doesn't count, continue trying with remaining darts
- Success: all subsequent darts in that leg count normally

### Out Strategy Details
- Applies only to final dart that reduces score to zero
- **Double Out**: Must be double segment (including double bull)
- **Master Out**: Must be double or triple segment
- **Straight Out**: Any segment acceptable

## Scoring Values
- **Single**: Face value (1-20)
- **Double**: 2 × face value (outer ring)
- **Triple**: 3 × face value (inner ring)
- **Bullseye**: 25 (outer) or 50 (inner, counts as double)
