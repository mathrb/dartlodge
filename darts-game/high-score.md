# High Score

## Overview
A simple game where players aim to score as many points as possible in a set number of darts. The player with the highest score wins.

## Rules
- **Darts per Turn**: Set number of darts (e.g., 9)
- **Objective**: Score as many points as possible
- **Winning**: The player with the highest score wins

## Scoring
- **Single**: Face value
- **Double**: Double the face value
- **Triple**: Triple the face value
- **Bullseye**: 25 points (outer bull) or 50 points (inner bull)

## Variations
- **Double High Score**: Only doubles count
- **Triple High Score**: Only triples count

## Implementation Notes
- Simple scoring accumulation
- Fixed number of darts per game
- Good for quick practice sessions
