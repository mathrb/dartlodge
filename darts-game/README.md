# Darts Game Modes

This folder contains detailed documentation for each darts game mode that can be implemented in the app.

## Available Game Modes

### Core Game Modes
1. **[X01 (Standard Darts)](x01.md)** - The classic darts game format (301, 501, 701, etc.) with configurable checkout strategies
2. **[Cricket](cricket.md)** - Strategic game closing numbers 15-20 and bullseye
3. **[Around the Clock](around-the-clock.md)** - Hit numbers 1-20 in sequence
4. **[Killer](killer.md)** - Eliminate opponents by hitting their assigned numbers
5. **[Shanghai](shanghai.md)** - Score on specific numbers over 7 rounds

### Additional Game Modes
6. **[Halve-It](halve-it.md)** - Avoid losing half your score by hitting target numbers
7. **[Baseball](baseball.md)** - Baseball-inspired scoring over 9 innings
8. **[Golf](golf.md)** - Complete each "hole" in fewest darts
9. **[Blind Killer](blind-killer.md)** - Killer variant with unknown opponents' numbers
10. **[Chase the Dragon](chase-the-dragon.md)** - Hit specific segments in sequence

### Variations and Special Modes
11. **[Scram](scram.md)** - Fast-paced scoring on specific numbers
12. **[High Score](high-score.md)** - Simple high score competition
13. **[Blind Cricket](blind-cricket.md)** - Cricket with unknown target numbers
14. **[Blind Shanghai](blind-shanghai.md)** - Shanghai with unknown target numbers
15. **[Blind Golf](blind-golf.md)** - Golf with unknown target numbers

## Implementation Priority

1. **501** - Essential for any darts app
2. **Cricket** - Very popular strategic game
3. **Around the Clock** - Great for beginners
4. **Killer** - Fun multiplayer game

The other modes can be added as the app grows in popularity and user base.
