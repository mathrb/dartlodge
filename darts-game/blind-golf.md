# Blind Golf

## Overview
A variation of the Golf game where players do not know which numbers are in play. The goal is to complete each "hole" in as few darts as possible.

## Rules
- **Holes**: Several holes, focusing on unknown numbers
- **Objective**: Complete each hole in as few darts as possible
- **Darts per Turn**: Unlimited darts per hole
- **Scoring**: The number of darts thrown to hit the number of the hole is recorded. The player with the lowest total number of darts after all holes wins
- **Winning**: The player with the lowest total score after all holes wins

## Scoring
- **Single**: Counts as a hit
- **Double/Triple**: Also counts as a hit

## Variations
- **Team Blind Golf**: Players form teams, and the goal is to complete each hole in as few darts as possible

## Implementation Notes
- Randomly select target numbers for each hole
- Hide target numbers from players
- Track darts thrown per hole
- Reveal target number after hole is completed
- Golf-themed scoring (lower is better)
