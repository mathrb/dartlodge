# Blind Shanghai

## Overview
A variation of the Shanghai game where players do not know which numbers are in play. The goal is to score as many points as possible by hitting specific numbers in sequence.

## Rules
- **Rounds**: Several rounds, focusing on unknown numbers
- **Objective**: Score as many points as possible by hitting the number of the round
- **Darts per Turn**: 3 darts
- **Scoring**: Only hits on the number of the round count. Doubles and triples score accordingly
- **Winning**: The player with the highest total score after all rounds wins

## Scoring
- **Single**: Face value
- **Double**: Double the face value
- **Triple**: Triple the face value

## Variations
- **Team Blind Shanghai**: Players form teams, and the goal is to score as many points as possible

## Implementation Notes
- Randomly select target numbers for each round
- Hide target numbers from players until round starts
- Only score hits on current round's target number
- Reveal target number after round completes
