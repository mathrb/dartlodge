# Scram

## Overview
A fast-paced game where players aim to hit specific numbers to score points. The game is played over several rounds, with each round focusing on a specific number.

## Rules
- **Rounds**: Several rounds, focusing on specific numbers
- **Objective**: Score as many points as possible by hitting the number of the round
- **Darts per Turn**: 3 darts
- **Scoring**: Only hits on the number of the round count. Doubles and triples score accordingly
- **Winning**: The player with the highest total score after all rounds wins

## Scoring
- **Single**: Face value
- **Double**: Double the face value
- **Triple**: Triple the face value

## Variations
- **Reverse Scram**: Start from 20 and go down to 1

## Implementation Notes
- Track current round and target number
- Only score hits on current round's target
- Fast-paced scoring game
