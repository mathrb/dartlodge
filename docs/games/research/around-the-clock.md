# Around the Clock

## Overview
A simple game where players aim to hit each number on the dartboard in sequence from 1 to 20. The first player to hit all numbers in order wins.

## Rules
- **Objective**: Hit numbers 1 through 20 in sequence
- **Darts per Round**: 3 darts
- **Order**: Players must hit the numbers in order (1, 2, 3, ..., 20)
- **Winning**: The first player to hit all numbers in order wins

## Scoring
- **Single**: Counts as a hit
- **Double/Triple**: Also counts as a hit but does not provide any additional advantage

## Variations
- **Reverse Around the Clock**: Start from 20 and go down to 1
- **Around the Clock Doubles**: Players must hit doubles for each number

## Implementation Notes
- Track current target number for each player
- Simple progression logic
- Good beginner game to implement first
