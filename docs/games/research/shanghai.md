# Shanghai

## Overview
A game where players aim to score as many points as possible by hitting specific numbers in sequence. The game is played over 7 rounds, with each round focusing on a specific number.

## Rules
- **Rounds**: 7 rounds, focusing on numbers 1 through 7
- **Objective**: Score as many points as possible by hitting the number of the round
- **Darts per Turn**: 3 darts
- **Scoring**: Only hits on the number of the round count. Doubles and triples score accordingly
- **Winning**: The player with the highest total score after 7 rounds wins

## Scoring
- **Single**: Face value
- **Double**: Double the face value
- **Triple**: Triple the face value

## Variations
- **Reverse Shanghai**: Start from 20 and go down to 14
- **Shanghai Cricket**: Combine elements of Shanghai and Cricket

## Implementation Notes
- Track current round number
- Only score hits on the current round's target number
- Simple scoring accumulation
